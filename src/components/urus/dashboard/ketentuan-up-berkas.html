Alur unggah berkas
  1. Unggah berkas dalam bentuk PDF.
  2. Unggah berkas dimulai setelah jadwal disetujui sampai maksimal H-8 jam rapat verifikasi.
    • Lewat batas waktu, proses verifikasi akan dibatalkan dan harus mendaftar ulang.
  3. Perhatikan tempat mengunggah berkas sesuai tabel di bawah ini:
    Dengan Jenis Folder
    FOLDER MERAH: silakan buat folder sendiri jika belum ada.
  4. Feedback akan diberikan lewat comment di berkas PDF dan secara langsung saat rapat verifikasi.
    • Konfirmasi revisi akan dikabarkan via email.
    • Apabila tidak ada revisi lagi, berkas yang sudah di-TTD akan dikirimkan via email dan proses verifikasi dinyatakan selesai.
  5. Unggah berkas revisi maksimal H+2 hari setelah diberi feedback.
    • Lewat batas waktu, proses verifikasi akan dibatalkan dan harus mendaftar ulang.
    • Maksimal bisa kurang dari H+2 hari (sebelum unggah berkas DPM.)
  6. Jangan lupa konfirmasi email setelah selesai mengunggah untuk notifikasi LEM/DPM.
