# kmfkuii-temp

## Installation

Clone the project

```bash
  git clone https://github.com/dpmfkuii/kmfkuii-temp.git
```

Go to the project directory

```bash
  cd kmfkuii-temp
```

Install the project

```bash
  npm run init
```

## Development

Start development watch

```bash
  npm run dev
```

Start Firebase emu

```bash
  npm run serve
```
